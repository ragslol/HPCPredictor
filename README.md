# HPCPredictor

This project trains two models: a linear regression model and a neural network on different High Performance Computing (HPC) metadata. It then uses that metadata to predict how long a job that is submitted is going to run


# How to Run
1. Download both the ML Final Project file and the clusterData.csv
2. Select a GPU to run the colab file and then click the run all button
3. The third cell will require you to click "Choose File" and select the clusterData.csv that you downloaded
